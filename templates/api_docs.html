EcoIQ REST API

Programmatic access to EcoIQ's climate intelligence — company scores, country profiles, portfolio screening, and financing compatibility signals. All endpoints return JSON. Authentication via Bearer token.

Authentication #

All API requests (except the public company list) require a Bearer token in the Authorization header. Tokens are issued per plan and rotate annually. To request a token, email alizhan@ecoiq.uk with your plan name and intended use case.

⚠ Never expose your API token in client-side code, public repositories, or shared documents. Rotate immediately at alizhan@ecoiq.uk if compromised.
HTTP Authorization header — all authenticated requests 
Authorization: Bearer eiq_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
Content-Type: application/json
Accept: application/json
cURL Authenticated request example 
curl -X GET "https://api.ecoiq.uk/v1/companies/schneider-electric/" \
  -H "Authorization: Bearer eiq_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -H "Accept: application/json"

Rate Limits #

Rate limits are enforced per API token, per calendar month. When you exceed your monthly limit, subsequent requests return 429 Too Many Requests with a X-RateLimit-Reset header indicating UTC reset time.

Plan Monthly calls Calls / second Concurrent Endpoints
Explorer ——— Public list only
Analyst 1,0002 req/s2 All except portfolio
Institutional 10,00010 req/s5 All endpoints
Enterprise Unlimited50 req/s20 All + custom

Error Codes #

EcoIQ API errors return a consistent JSON body alongside the HTTP status code. All errors include a machine-readable code and a human-readable message.

JSON Error response shape 
{
  "error": {
    "code":    "rate_limit_exceeded",
    "message": "Monthly API quota exhausted. Resets 2026-06-01T00:00:00Z.",
    "status":  429,
    "docs":   "https://ecoiq.uk/api-docs/#rate-limits"
  }
}
StatusCodeMeaning
200okRequest succeeded
400invalid_paramsMissing or malformed query parameters
401unauthorizedMissing or invalid Bearer token
403plan_restrictedEndpoint requires a higher plan tier
404not_foundCompany or country slug not found
422validation_errorPOST body failed validation (see errors array)
429rate_limit_exceededMonthly or per-second rate limit hit
500internal_errorUnexpected server error — contact support
503service_unavailablePlanned maintenance — check status page

Python — requests library

Python Quick-start helper 
import requests

class EcoIQClient:
    BASE = "https://api.ecoiq.uk/v1"

    def __init__(self, token):
        self.session = requests.Session()
        self.session.headers.update({
            "Authorization": f"Bearer {token}",
            "Accept": "application/json",
        })

    def companies(self, sector=None, min_score=None):
        params = {}
        if sector:    params["sector"]    = sector
        if min_score: params["min_score"] = min_score
        return self.session.get(f"{self.BASE}/companies/", params=params).json()

    def company(self, slug):
        return self.session.get(f"{self.BASE}/companies/{slug}/").json()

    def financing(self, slug):
        return self.session.get(f"{self.BASE}/financing/{slug}/").json()

# Usage
client = EcoIQClient("eiq_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxx")
profile = client.company("schneider-electric")
print(profile["ecoiq_score"])  # 85.2

Score Schema #

The EcoIQ score is a weighted composite (0–100) of six pillars with a harm penalty applied last. All sub-scores are also 0–100 before weighting.

ecoiq_scorefloatComposite score 0–100. 1 decimal place. Harm penalty already deducted.
environmental_responsibilityfloatWeight 30%. Emissions, pollution, energy transition, climate risk management.
modernizationfloatWeight 20%. Technology upgrade, clean capex, circular economy, R&D spend.
ethics_governancefloatWeight 15%. Anti-corruption, audit quality, transparency, board independence.
public_benefitfloatWeight 15%. Employment quality, community impact, regional development.
future_readinessfloatWeight 20%. Net-zero alignment, green bond eligibility, regulatory readiness.
harm_penaltyfloatNegative value (0 to –30). Applied for severe pollution, governance failures, and public benefit extraction.
evidence_coverageinteger0–100%. Percentage of scoring dimensions backed by verifiable public evidence.
ai_confidenceinteger0–100. EcoIQ AI confidence in the profile accuracy. Drives confidence label.
confidence_labelstringOne of: low, medium, high, verified.

Full methodology: ecoiq.uk/methodology/ →
