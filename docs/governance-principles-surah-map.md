# EcoIQ Capital Ethics Compendium — Internal Surah Mapping

**INTERNAL ONLY — NOT FOR PUBLIC DISPLAY, API RESPONSES, OR MARKETING**

This document maps the 114 Surahs of the Qur'an to the corresponding
EcoIQ Capital Ethics Compendium governance principles. The public-facing
page at `/governance-principles/` uses professional English principle names only.
No Surah names, Arabic terminology, or Qur'anic references appear in any
public-facing code, API response, or marketing material.

Refer to: `docs/platform-overview.md` (Module 05 language guidance section)
for the full list of approved and prohibited language.
