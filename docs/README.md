## Tazkiyah 114 — The Surah Map

A Qur'an-inspired reflection and action platform: **114 Surahs → 114 life pathways.** It helps
people move from a real-life struggle to Qur'anic guidance, reflection, action, dua, journaling,
and long-term tazkiyah.

> **Tazkiyah 114 is currently a preview/prototype. All religious guidance content requires
> sources, tafsir references, qualified scholar review, and wellbeing review where sensitive
> before public educational use.**
