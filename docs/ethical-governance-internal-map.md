## Internal Scoring Note

The dimension weights in the EcoIQ Ethical Finance Fit module (Module 05)
are informed by the Maqasid al-Shariah framework:

| Maqasid Objective | Internal Weight Rationale |
|---|---|
| Hifz al-Nafs (life / protection) | Highest weight in harm and safety dimensions |
| Hifz al-Mal (wealth / fair capital) | Governs capital structure and profit distribution |
| Hifz al-'Aql (intellect / governance) | Underlies oversight and transparency scores |
| Hifz al-Nasl (progeny / future) | Drives intergenerational and long-term dimensions |
| Hifz al-Din (faith / ethical alignment) | Shapes excluded-activity screening |

- Any public URL or page
- Any API response field or key
- Any email template
- Any marketing, press or investor material
- Any navigation label
