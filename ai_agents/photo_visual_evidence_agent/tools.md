# Photo / Visual Evidence Agent — Tools

## EcoIQ modules this agent reads from / writes to

- **Mobile / iPad Inspection Mode** — primary capture source for photos/videos
- **Omnimodal Evidence Panel** — combines this agent's output with other evidence types
- **Asset Passport** — receives photo-derived condition notes
- **Data Room & Evidence Vault** — stores the source imagery
- **Knowledge Graph & Relationship Map** — evidence nodes linked to the asset
- **Security, Privacy & Compliance Centre** — governs handling of any detected PII (faces, plates)

## External tool concepts (not yet wired to a live runtime)

- Vision-capable model for image/video description
- Basic gauge/display OCR for reading visible numeric displays only

## Explicit non-tools

- No automated safety-compliance certification
- No automated root-cause diagnosis
- No image alteration/enhancement that could misrepresent the original evidence
