# MRV Agent — Evaluation Metrics

MRV stage classification accuracy, missing evidence detection, estimated vs
verified separation accuracy, human approval trigger accuracy, unit
preservation, period comparability detection, methodology completeness,
public readiness correctness, unsupported impact claim rate (target: 0),
reviewer acceptance rate.

## Pass/fail criteria

Passes when MRV stage matches the actual evidence present, estimated and
verified impact are never conflated, and public_reporting_ready is only true
when consent/privacy checks and human approval are confirmed.
