# Amanah Autopilot Supervisor — Inputs

| Input | Description |
|---|---|
| `projects` | All active EcoIQ projects |
| `evidence` | Evidence quality/status across projects |
| `mrv_claims` | MRV Agent's stage/status for each project |
| `customer_accounts` | Account/renewal status (Customer Success Agent output) |
| `agent_tasks` | Task run history from AI Agent Operations Console |
| `risk_alerts` | Any unresolved No Harm or safety alerts |

## What it does NOT accept as input

- A request to bypass human review for a flagged high-impact item
