# Report Generator Agent — Demo Scenarios

## Investor demo scenario

**Audience:** investor receiving an investor memo for an MRV Verified project.

Report Generator Agent produces a memo where every figure links back to its
MRV claim, finance model, or governance approval — the investor can click
through to source evidence for any statement in the document.

## Enterprise / government use case

**Audience:** a government partner requesting a country brief across a
portfolio of pilot projects.

Report Generator Agent assembles a consistent brief structure (executive
summary, risks, assumptions, next action, evidence links) across every
project, making cross-project comparison possible for policy decisions.

## Islamic ethical finance / Amanah alignment

Any Maqasid/Mizan or Islamic finance wording in a report only appears after
Governance Agent's review has approved it — Report Generator Agent carries
the approved wording forward, it does not originate ethical claims.

## MRV / audit trail requirements

`evidence_links` in every report is the audit trail an external reviewer or
regulator would need to verify every claim independently, back to its
original source document or approval record.
