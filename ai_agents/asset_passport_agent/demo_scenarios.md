# Asset Passport Agent — Demo Scenarios

## Investor demo scenario

**Audience:** sovereign fund reviewing a portfolio of industrial assets for
transition finance readiness.

Asset Passport Agent assembles a single-page, evidence-linked record for
Boiler House #3 — showing exactly what is known, what is missing, and one
clear next step ("collect 12 months of fuel data"), rather than a polished
but unsupported summary.

## Enterprise / government use case

**Audience:** a national industrial ministry standardising asset records
across hundreds of municipally-owned facilities.

Asset Passport Agent gives every facility the same structured record shape,
making cross-site comparison possible for the first time — while keeping
each field's evidence trail visible to auditors.

## Islamic ethical finance / Amanah alignment

Where evidence indicates a community-health-relevant condition, the passport
carries a note forward to Governance Agent rather than resolving it — keeping
Maqasid-aligned harm review a human decision.

## MRV / audit trail requirements

Every baseline field in the passport is the exact evidence MRV Agent needs to
establish a baseline period — so passport quality directly determines how
fast a project can reach MRV Verified status.
