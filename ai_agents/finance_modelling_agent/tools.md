# Finance Modelling Agent — Tools

## EcoIQ modules this agent reads from / writes to

- **Institutional Finance Engine** — the platform module this agent's output feeds
- **Industrial Playbook Matching Agent** — supplies the actions being costed
- **Supplier & Funding Marketplace** — receives the funding gap/route needed
- **Governance & Expert Review Board** — financial reviewer sign-off
- **Executive Briefing & Board Pack Generator** — consumes the finance memo draft

## External tool concepts (not yet wired to a live runtime)

- Standard payback/NPV/IRR calculation library
- Currency and tariff lookup by country

## Explicit non-tools

- No automated approval of a finance model for external use
- No Shariah-compliance certification
