# Finance Modelling Agent — Demo Scenarios

## Investor demo scenario

**Audience:** Islamic finance provider evaluating a clean heating pilot.

Finance Modelling Agent produces a draft CAPEX/OPEX model with every
assumption visible, an estimated (not guaranteed) payback period, and a
factual description of murabaha/ijara structuring options — with a clear
note that Shariah compliance requires separate qualified review.

**Why it lands:** investors see the model's assumptions, not a black box —
and the ethical-finance framing is handled honestly rather than oversold.

## Enterprise / government use case

**Audience:** a national development bank comparing funding gaps across a
portfolio of industrial retrofit projects.

Finance Modelling Agent's consistent output schema lets the bank rank
projects by funding gap size and risk profile across dozens of sites in one
comparable table.

## Islamic ethical finance / Amanah alignment

Every Islamic finance structuring description is factual and clearly
separated from a compliance determination — supporting rizq without zulm
(wealth creation without exploitation) as a design principle, not a marketing claim.

## MRV / audit trail requirements

Every cost figure links back to its source (bill, quote, assumption) so a
financial reviewer or MRV auditor can trace the entire model back to
evidence at any time.
