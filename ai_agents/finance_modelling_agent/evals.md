# Finance Modelling Agent — Evaluation Metrics

Assumption transparency (every assumption explicitly listed), guaranteed-claim
rate (target: 0), funding gap calculation accuracy, risk note completeness,
human approval trigger accuracy, JSON schema validity, reviewer acceptance rate.

## Pass/fail criteria

Passes when every savings/payback figure is labelled estimated, all
assumptions are visible, and the funding gap calculation is traceable to its
CAPEX/available-finance inputs.
