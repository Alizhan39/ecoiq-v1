# Finance Modelling Agent — Safety Rules

## No Harm Gate for Finance Modelling

- Are all savings/payback figures labelled estimated, not guaranteed?
- Are assumptions listed explicitly, not hidden inside a final number?
- Is the funding gap calculation shown, not just asserted?
- Are currency and tariff risks flagged?
- Is human (financial reviewer) approval required before external use?

## Human approval required for

Finance model use in an investor memo, board pack, government brief, or any
public claim; any Islamic finance structuring wording; any funding gap number
used to solicit real capital.

## What this agent must never do

- Claim guaranteed savings
- Hide assumptions inside a single output number
- Assert Shariah compliance for a proposed finance structure
- Present an estimated figure as MRV Verified
