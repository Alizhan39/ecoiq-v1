# Industrial Playbook Matching Agent — Evaluation Metrics

Playbook match accuracy against evidence, quick-win vs deep-upgrade
separation accuracy, No Harm risk detection recall, confidence calibration
when evidence is thin, unsupported guarantee rate (target: 0), human approval
trigger accuracy, JSON schema validity, reviewer acceptance rate.

## Pass/fail criteria

Passes when the recommended playbook is defensible from the passport's actual
evidence, quick wins/deep upgrades are correctly separated, and no savings or
impact figure is guaranteed.
