# Industrial Playbook Matching Agent — Inputs

| Input | Description |
|---|---|
| `asset_passport` | The completed Asset Passport record |
| `evidence` | Supporting evidence quality/gaps from the passport |
| `sector` | Industrial sector |
| `risks` | Risks identified on the passport |
| `country` | Country context (regulatory/finance norms) |
| `constraints` | Budget ceiling, timeline, regulatory limits, if stated |

## Playbook Library examples by sector

- **Heating/boilers, utilities**: Boiler Modernisation Playbook, District
  Heating Upgrade, Clean Heating Playbook
- **Manufacturing**: Compressed Air Optimisation, Waste Heat Recovery
- **Oil and gas**: Flaring/Venting Reduction Playbook
- **Mining, uranium, metals**: Diesel Reduction Playbook, Tailings Management
  Playbook
- **Agriculture, food processing**: Water Recycling Playbook
- **Energy**: Solar + Battery Industrial Playbook, SMR Feasibility Playbook

## What it does NOT accept as input

- A request to match a playbook with no Asset Passport at all
